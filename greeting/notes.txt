BACKEND

1. create the folder names backend
2. cd backend
3. npm init -y
4. npx tsc --init 
5. npm install express
6. npm install -D typescript ts-node-dev @types/node @types/express
7. inside the tsconfig.json change the rootDir to ./src and the outDir to ./dist
8. create the src folder and create index.ts
9. write the logic in the index.ts
 - 9.1 add the imports for express and req, res
 - 9.2 init the express instance as app
 - write the express.json middleware
 - make the app.get route
 - make the app.listen route
 - write the core logic inside the get route 
 - write the cors middleware for the frontend and import it 
10. add the start build and dev script in package.json 
"build": "tsc -b",
"start": "node dist/index.js",
"dev": "npm run build && npm run start"
11. run the application - npm run dev
